Exercises

Exercise 6: Clap Switch

INMP441 • Toggle

Listens for 1 clap → GREEN LED ON / 2 claps → GREEN LED OFF. Includes warmup, spike detection, and re-arm logic to avoid false triggers.

Input: INMP441 I2S microphone Output: GREEN LED (D6)
Ready

Exercise 7: Sound Level Indicator

INMP441 • 3 Levels

Shows sound intensity: solid GREEN = quiet, running G→O→R pattern = moderate, solid RED = loud. Uses adaptive thresholds.

Input: INMP441 I2S microphone Output: LEDs (D5 red, D6 green, D13 orange)
Ready

Exercise 8: Noise Level Classifier

INMP441 • 3 States

Real-time noise classification with hold logic to prevent rapid LED flickering. GREEN = quiet, ORANGE = moderate, RED = noisy.

Input: INMP441 I2S microphone Output: LEDs (D5 red, D6 green, D13 orange)
Ready

Exercise 9: Voice Activated System

INMP441 • Voice + LCD

Activates on sustained voice/speech: updates LCD message and beeps buzzer (2 beeps). Uses adaptive threshold and voice duration check.

Input: INMP441 I2S microphone Output: LCD + Buzzer (D21)
Ready

Exercise 10: Loud Sound Alert

INMP441 • Alarm

Triggers alarm (buzzer + blinking red/orange LEDs) when loud sound is detected for a minimum duration. Features cooldown and adaptive threshold.

Input: INMP441 I2S microphone Output: Buzzer (D21) + Red LED (D5) + Orange LED (D13) + optional LCD
