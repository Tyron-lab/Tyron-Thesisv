Exercise 16: PIR → Servo Spin

PIR • Servo

Motion detected → servo spins fast; no motion → total stop (pulses off).

Exercise 17: Voice LED Control

Mic • LEDs

Speech commands (open/close + color) control red (D5), orange (D13), green (D6) LEDs. Uses Vosk model with grammar.

Input: Microphone (speech) Output: RGB LEDs

Exercise 18: Clap Relay Sequence

Mic • Relays

Clap toggles pattern: relays cycle ON/OFF sequentially (CH1→CH2→CH3→CH4). Active-low relays with threshold and cooldown.

Input: Microphone (clap) Output: 4 Relays (D27, D10, D26, D25)
Ready

Exercise 19: PIR Intrusion Alert

PIR • LED • Buzzer • LCD

Motion → red LED ON, buzzer beeps (0.15s on/0.12s off), LCD "INTRUDER ALERT"; no motion → all off, LCD "SECURE".

Input: PIR sensor (D22) Output: Red LED (D5), Buzzer (D21), LCD
Ready

Exercise 20: Ultrasonic Smart Response

HC-SR04 • Servo • LED • Buzzer

Near (<=20cm) → servo 90° hold, orange ON, buzzer beep (0.1s on/off); far → servo 0° (off), all off. Reads every 0.12s.

Input: Ultrasonic (D23 trig, D24 echo) Output: Servo (D12), Orange LED (D5), Buzzer (D21)
